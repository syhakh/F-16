# F-16  
#### 情感文本分类模型的Web网站  
----------  
#### 环境搭建  
>1. python==3.6  
>2. tensorflow==1.14.0  
>3. Keras==2.2.4  
>4. Window环境，linux尚未解决端口被占用的报错  
>5. Falsk==1.1.1
----------  
#### 运行步骤  
>1.首先运行app.py,app.py包含分类模型的搭建与模型预测结果向前端网页传送的两个部分。  
>2.然后运行login.html，进行文本输入，点击查看预测标签。
